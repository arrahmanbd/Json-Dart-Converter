Flutter REST API Tutorial

January 1, 2021 by AR

In this section, we are going to learn how we can access the REST API in the Flutter app. Today, most of the apps use remote data using APIs. So, this section will be the important part for those developers who want to make their carrier in Flutter.

Example is better then tons of words. In this example I am going to fetch data from this website. Take a look of our json data we want to fetch.
Todo API

Add HTTP package and Internet Permission

First of all create a new project. To install the http package, open the pubspec.yaml file in your project folder and add http package in the dependency section. You can get the latest http package here and add it like 

        
            dependencies:  
             http:

Generate Model Class

Again, goto this url Todo API and copy full response data. Now go to our JSON to dart Model Generator, give it a name (ie: TodoModel) and Generate Model class

Make a network request by using the http package

In this step, you need to make a network request by using the http.get() method, as shown below, we make a network request which is a Future function with our model type. 

        
import 'dart:convert';
import 'package:http/http.dart' as http;

import '../models/todo_model.dart';

String url = 'https://jsonplaceholder.typicode.com/todos';

Future getTodo() async {
  final response = await http.get(Uri.parse(url'));
  var data = jsonDecode(response.body.toString());
  if (response.statusCode == 200) {
    return TodoModel.fromJson(data);
  } else {
    throw Exception('Failed to load');
  }
}

Basically its requesting for json data in TodoAPI server and convert the response getting from network request into our custom Model Class previously we generate.

Conclusion

In this article, we were introduced to REST API (which is short for REpresentational State Transfer), which, if we put it in layman's language, is responsible for fetching data from the internet and putting it in our flutter application so that the users/clients can enjoy some of the additional features implemented by the developer.
